# merge-3-collections

Merge three integer arrays into one ascending-sorted array without using any sort function.

Assumptions:
- collection_1 is sorted ascending
- collection_2 is sorted ascending
- collection_3 is sorted descending

## Setup

```bash
npm install
```

### Excute the Code
    Run The typescript directly

    Build
```bash
  npm run build
```

```bash
  npm run dev
```
    Test
```bash
  npm test
```
